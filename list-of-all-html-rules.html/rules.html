To write a good HTML document, there are rules that you have to follow.

These are some of them:
  • You cannot have an end tag without a start tag
  • Every start tag must start with a greater than symbole and end with a less than symbole
  • Every end tag must start with a greater than symbole followed by a / and end with a less than symbole
  • Use lower case for the best of your code
