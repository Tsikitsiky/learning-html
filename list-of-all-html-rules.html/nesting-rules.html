The nesting rules

What is nesting?

Nesting involves putting HTMLelements within other elements in the proper order. The elements that are "inside" of other elements are known as nested elements, and they are essential to building any webpage today.

Rules

To have a well nested elements, follow these rules:
  • make sure that you know the parent elements
  • don't do this!
    <p>This<b>word</p>is bold</b>
    instead do this
    <p>This<b>word</b>is bold</p>
  • indent your code properly so that you know which is child or sibling elements
